# 1BNWL

[1BNWL(1-Bit Non-Warpping Language)](https://esolangs.org/wiki/1BNWL) is a bf derivative with 1-bit tape by [ChuckEsoteric08](https://esolangs.org/wiki/User:ChuckEsoteric08) in 2022

Turing-complete if tape is infinite

## h

prints 1001000 which is h in binary

```
+.>..<.>...
```

## 1110

prints 1110

```
>+>+>+[.<].
```

### How to Run

already built, if need build 

```
go build
```

```
esolang-implementation-project h.bnwl
```
